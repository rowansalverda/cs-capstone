# AUVIC Website Redesign Project

## Project Overview

This project is a complete redesign and redevelopment of the AUVIC (Autonomous Underwater Vehicles Club at the University of Victoria) website. The goal of the project was to modernize the club’s online presence, improve navigation and accessibility, create a more consistent visual identity, and better showcase the team’s vehicles, competitions, sponsors, and members.

The website was built from scratch using HTML, CSS, and JavaScript, with a focus on responsive design, smooth animations, and improved user experience across desktop and mobile devices.

---

## Features

- Responsive website design for desktop and mobile devices
- Modern navigation bar with dropdown menus
- Homepage hero section with video background
- Interactive 3D submarine viewer
- Vehicle showcase pages for multiple submarines
- Competition pages with scrolling participant sections
- Team and admin profiles with interactive pop-ups
- Sponsor showcase section
- Sticky navigation and call-to-action elements
- Contact page with countdown timer and social links
- Mobile-friendly layouts and animations
- Consistent branding and typography throughout the site

## Installation Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/your-repository-name.git
```

### 2. Open the Project Folder

```bash
cd AUVIC
```

### 3. Open the Website

You can open the project using:

- Visual Studio Code with the Live Server extension
- Any local development server
- A standard web browser (some features such as the 3D viewer may require a local server)

Example using VS Code Live Server:

1. Open the project folder in Visual Studio Code
2. Install the **Live Server** extension
3. Right-click `index.html`
4. Select **Open with Live Server**

## Usage

- Open `index.html` to access the homepage
- Use the navigation bar to explore:
  - About Us
  - Competitions
  - Vehicles
  - Sponsors
  - Contact Information
- View vehicle galleries and competition details
- Interact with the 3D submarine viewer
- Access social media and Discord links through the contact page
